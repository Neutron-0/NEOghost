
dotnet build "NeoGhost\NeoGhost.sln" -c Release

The Inno installer packages the full Release output folder:
NeoGhost\bin\Release\net10.0-windows10.0.17763.0\win-x64

Do not copy only NeoGhost.exe into the installer. The application also needs
NeoGhost.dll, NeoGhost.runtimeconfig.json, NeoGhost.deps.json, NeoGhost.dll.config,
and dependency DLLs from the same output folder.
